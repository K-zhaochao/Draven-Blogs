### 泛型

![[Draven_Note_Images/Java/15.集合/集合的进阶/集合的进阶_泛型_24.png]]

如果没有给集合增加泛型都话，就可以给集合添加任意的引用数据类型的元素。

但是因为多态的弊端是不能调用子类的特有功能，所有如果不给集合限定泛型都话就无法在遍历的过程中统一调用该类对应的特有功能。

    public static void main() {
        ArrayList<Ye> list1 = new ArrayList<>();

    }

    }

}

}

}

### 不可变集合

#### 创建方法：

![[Draven_Note_Images/Java/15.集合/集合的进阶/集合的进阶_创建方法：_83.png]]

1. 对于创建Set的不可变集合，传入的参数不能重复。
2. 对于创建Map的不可变集合，传入的键不能重复，以及最多传入10个键值对。
3. 想要创建超过10个数据的Map不可变集合就需要用到`ofEntry()`的方法，传入一个含有entry的数组

![[Draven_Note_Images/Java/15.集合/集合的进阶/集合的进阶_创建方法：_84.png]]

**注意：** 如果用的JDK版本>=10，那么直接用copyOf()方法将HashMap集合传入即可获得不可变的Map集合。

#### 方法一：继承 `Thread` 类

1. 先创建一个类继承 `Thread` 类，且在该类中重写 `run` 方法【该方法中运行该线程中的操作】
``` java
public class ThreadDemo extends Thread {  
      
    private Integer number;  
      
    public ThreadDemo(Integer number) {  
        this.number = number;  
    }  
      
    @Override  
    public void run() {  
        System.out.println(number);  
    }  
}
```

2. 只需在测试类中创建刚才的实体类，再调用 `start` 方法即可运行之前重写的 `run` 方法。
``` java
static void main() {  
    ThreadDemo threadDemo = new ThreadDemo(20);  
    threadDemo.start();  
}
```

#### 方法二：实现 `Runnable` 接口

1. 让实体类实现 `Runnable` 接口，并在该类中重写 `run` 方法。
``` java
public class ThreadDemo implements Runnable {  
  
    private final Integer number;  
  
    public ThreadDemo(Integer number) {  
        this.number = number;  
    }  
  
    @Override  
    public void run() {  
        // 创建当前执行的线程对象  
        Thread thread = Thread.currentThread();  
        System.out.println(thread.getName() + number);  
    }  
}
```

2. 只需在测试类中创建一个上面的实体类对象以及一个 `Thread` 对象【将实体类对象作为参数传入】并调用 `start` 方法即可开启线程。
``` java
static void main() {  
    // 创建实现类对象  
    ThreadDemo threadDemo = new ThreadDemo(20);  
    // 创建线程  
    Thread thread = new Thread(threadDemo);  
    // 设置线程名字  
    thread.setName("thread");  
    // 开启线程  
    thread.start();  
}
```

#### 方法三：利用 `Callable` 接口和 `Future` 接口方式实现
**注意：** 该种方法是用于解决线程没有返回值的问题。
1. 创建一个实体类实现 `Callable<H>` 接口中的 `call` 方法【该方法是有返回值的，表示多线程的运行结果，H表示返回值的类型】。
``` java
public class MyCallable implements Callable<Integer> {  
    @Override  
    public Integer call() throws Exception {  
        int sum = 0;  
        for (int i = 1; i <= 10; i++) {  
            sum += i;  
        }  
        return sum;  
    }  
}
```

2. 只需在测试类中创建实现类对象、 `FutureTask` 【 `Future` 的实现类，将实现类作为参数放入，该类可管理多线程的运行结果】对象和 `Thread` 对象【将 `FutureTask` 对象作为参数，表示该线程要执行的操作】，最后用  `FutureTask` 对象的 `get` 方法获取刚刚线程执行之后的结果。
``` java
static void main() throws ExecutionException, InterruptedException {  
    // 创建 Callable 的实现类对象 MyCallable
	MyCallable myCallable = new MyCallable();  
    // 创建 Future 的实现类对象 FutureTask 并将 Callable 的实现类对象作为参数  
    FutureTask<Integer> futureTask = new FutureTask<>(myCallable);  
    // 创建 Thread 对象并将 Future 的实现类对象作为参数放入  
    Thread thread = new Thread(futureTask);  
    // 启动线程  
    thread.start();  
    // 通过 Future 的实现类对象的get方法获取线程返回值  
    Integer result = futureTask.get();  
    System.out.println(result);  
}
// os:属实是套娃
```

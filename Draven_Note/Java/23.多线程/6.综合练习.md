### 题目四：

![[Draven_Note_Images/Pasted image 20260606205900.png]]

`MyThread` 类代码
``` java
public class MyThread extends Thread {  
  
    // 红包金额100，分成3份  
    static BigDecimal money = BigDecimal.valueOf(100.0);  
  
    // 红包个数  
    static int count = 3;  
  
    // 最小的中奖金额  
    static final BigDecimal MIN = BigDecimal.valueOf(0.01);  
  
    @Override  
    public void run() {  
        // 同步代码块  
        synchronized (MyThread.class) {  
            if (count == 0) {  
                // 判断，共享数据是否到了末尾（已经到末尾）  
                System.out.println(getName() + "没有抢到红包。");  
            } else {  
                BigDecimal prize;  
                // 判断，共享数据是否到了末尾（没有到末尾）  
                if (count == 1) {  
                    // 表示这是最后一个红包  
                    prize = money;  
                } else {  
                    //获取抽奖范围  
                    double bounds = money.subtract(BigDecimal.valueOf(count-1).multiply(MIN)).doubleValue();  
                    Random r = new Random();  
                    //抽奖金额  
                    prize = BigDecimal.valueOf(r.nextDouble(bounds));  
                }  
                //设置抽中红包，小数点保留两位，四舍五入  
                prize = prize.setScale(2, RoundingMode.HALF_UP);  
                // 从money当中，去掉当前中奖的金额  
                money = money.subtract(prize);  
                // 红包的个数-1  
                count--;  
                // 本次红包的信息进行打印  
                System.out.println(getName() + "抢到了" +  prize + "元红包！");  
            }  
        }  
    }  
}
```

#### **解法1：** 
`MyThread` 类代码
``` java
public class MyThread extends Thread {  
    ArrayList<Integer> list;  
  
    // 分别表示两个抽奖箱的抽奖结果  
    static ArrayList<Integer> list1 = new ArrayList<>();  
    static ArrayList<Integer> list2 = new ArrayList<>();  
  
    // 分别表示两个抽奖箱的最高奖项  
    static int MAX1 = 0;  
    static int MAX2 = 0;  
  
    // 分别表示两个抽奖箱的总计额  
    static int sumPrize1 = 0;  
    static int sumPrize2 = 0;  
  
    public MyThread(ArrayList<Integer> list) {  
        this.list = list;  
    }  
  
    @Override  
    public void run() {  
        while (true) {  
            synchronized (MyThread.class) {  
                if (list.isEmpty()){  
                    if ("抽奖箱1".equals(getName())) {  
                        System.out.println("抽奖箱1" + list1 + "最高奖项为：" + MAX1 + ", 总计额为：" + sumPrize1);  
                    } else {  
                        System.out.println("抽奖箱1" + list2 + "最高奖项为：" + MAX2 + ", 总计额为：" + sumPrize2);  
                    }  
                    break;  
                } else {  
                    // 打乱奖项  
                    Collections.shuffle(list);  
                    Integer prize = list.removeFirst();  
                    if ("抽奖箱1".equals(getName())) {  
                        sumPrize1 += prize;  
                        if (prize > MAX1) {  
                            MAX1 = prize;  
                        }  
                        list1.add(prize);  
                    } else {  
                        sumPrize2 += prize;  
                        if (prize > MAX2) {  
                            MAX2 = prize;  
                        }  
                        list2.add(prize);  
                    }  
                }  
            }  
            try {  
                Thread.sleep(10);  
            } catch (InterruptedException e) {  
                throw new RuntimeException(e);  
            }  
        }  
    }  
}
```

### 题目七：

![[Draven_Note_Images/Pasted image 20260606210118.png]]

`MyCallable` 类代码
``` java
public class MyCallable implements Callable<Integer> {  
    ArrayList<Integer> list;  
  
    public MyCallable(ArrayList<Integer> list) {  
        this.list = list;  
    }  
  
    @Override  
    public Integer call() throws Exception {  
        // 表示抽奖箱的抽奖结果  
        ArrayList<Integer> prizeList = new ArrayList<>();  
        // 表示抽奖箱的最高奖项  
        int MAX = 0;  
        // 表示抽奖箱的总计额  
        int sumPrize = 0;  
  
        while (true) {  
            synchronized (MyThread.class) {  
                if (list.isEmpty()){  
                    System.out.println(Thread.currentThread().getName() + prizeList + "最高奖项为：" + MAX + ", 总计额为：" + sumPrize);  
                    break;  
                } else {  
                    // 打乱奖项  
                    Collections.shuffle(list);  
                    Integer prize = list.removeFirst();  
                    sumPrize += prize;  
                    if (prize > MAX) {  
                        MAX = prize;  
                    }  
                    prizeList.add(prize);  
                }  
            }  
            try {  
                Thread.sleep(10);  
            } catch (InterruptedException e) {  
                throw new RuntimeException(e);  
            }  
        }  
        if (prizeList.isEmpty()) {  
            return null;  
        }  
        return MAX;  
    }  
}
```

测试类代码
``` java
public class Test {  
    static void main() throws ExecutionException, InterruptedException {  
        // 创建奖池  
        ArrayList<Integer> list = new ArrayList<>();  
        Collections.addAll(list, 10, 5, 20, 50, 100, 200, 500, 800, 2, 80, 300, 700);  
  
        // 创建多线程要运行的参数对象  
        MyCallable myCallable =  new MyCallable(list);  
  
        // 创建多线程运行结果的管理者对象  
        FutureTask<Integer> ft1 = new FutureTask<>(myCallable);  
        FutureTask<Integer> ft2 = new FutureTask<>(myCallable);  
  
        // 创建线程  
        Thread t1 = new Thread(ft1);  
        Thread t2 = new Thread(ft2);  
  
        // 设置线程名称  
        t1.setName("抽奖箱1");  
        t2.setName("抽奖箱2");  
  
        // 开启线程  
        t1.start();  
        t2.start();  
  
        // 获得线程奖项最大值  
        Integer max1 = ft1.get();  
        Integer max2 = ft2.get();  
  
        // 打印抽到最高奖项的奖箱  
        System.out.println("在此次抽奖过程中, " + (max1>max2?"抽奖箱1":"抽奖箱2") + "中产生了最大奖项" +  
                ", 该奖项的金额为: " + (max1>max2?max1:max2));  
    }  
}
```

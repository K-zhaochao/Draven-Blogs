# 附录3：关于IDEA

### 快捷键：

1. **`shift + F6`：** 批量更改变量名。
2. **`Fn+Alt+Insert`：** 快速生成构造方法/常用的方法（如get/set）。
3. **`Ctrl+shift+U`：** 将选中的英文字母从小/大写变成大/小写。
4. **`Ctrl+P`：** 用于查看该方法的参数列表。
5. **`Ctrl+F12`：** 可以看见类中所有的成员。
6. **`Alt+ENTER`：** 查看报错/警告行的具体信息。
7. **`Ctrl+N`：** 用于搜索Java包中的各种类。
8. **`Ctrl+Alt+←`：** 返回上一页。
9. **`Ctrl+Alt+V`：** 自动生成方法返回的变量。
10. **`Ctrl+Alt+T`：** 快速把选中的内容包裹进if/for/try等语句中。
11. **`Ctrl+Alt+L`：** 代码格式化。
12. **`Ctrl+Alt+m`：** 选中的代码块能重新快速生成一个方法。
13. **`Ctrl+O`：** 弹出继承者的方法，可选中进行重写。
14. **`Ctrl+Alt+B`：** 向下转型。
15. **`Ctrl+h`：** 查看该接口有哪些实现类。

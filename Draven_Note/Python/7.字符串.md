### 常用方法

![[Draven_Note_Images/Python/7.字符串/7. 字符串_常用方法_11.png]]

#### 查找元素：

`变量名.index(查找元素)` 返回对应元素的索引，如果要查找的元素也是一个字符串，则会返回其对应的第一个元素的索引。

#### 替换元素：

`字符串.replace(字符串1, 字符串2)` ，能将字符串内的字符串1修改为字符串2并返回一个新的字符串，并且之前的字符串不会被修改。例如：

``` python
str = "1245454512"   
new_str = str.replace("12", "00")
# 这样str中的全部"12"都会修改为"00"并返回修改后的字符串
```

#### 分割元素：

`字符串.split(分隔符字符串)`，按照指定的分隔符字符串将整个字符串分割成多个字符串然后存入到列表中，返回列表，被分割的字符串本身不发生改变。

#### 规整操作：

`字符串.strip()`可以去除字符串的前后空格和换行符；`字符串.strip(字符串)`可以去除字符串前后指定的字符串，注意传入的字符串表示的是需要移除的字符，最后会返回处理之后的字符串。

#### 元素个数：

`字符串.count(查找字符串)`返回字符串总查找字符串出现的个数。

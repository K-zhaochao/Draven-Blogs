# 12.来电提醒和客户催单

### WebSocket

#### 介绍：

![[Draven_Note_Images/Pasted image 20260520222540.png]]

#### 入门案例：

``` java
/**  
 * WebSocket配置类，用于注册WebSocket的Bean  
 */@Configuration  
public class WebSocketConfiguration {  
  
    @Bean  
    public ServerEndpointExporter serverEndpointExporter() {  
        return new ServerEndpointExporter();  
    }  
  
}
```
**注意：** 该配置类必须存在，当启动类加上 `@EnableWebSocket` 注解之后，`Spring Boot` 会自动开启并启动 `WebSocket` 服务端，并且它的存在会让 `Spring Boot` 去找到所有带着 `@ServerEndpoint` 注解的类。

``` java
Map map = new HashMap();  
map.put("type", 1); // 1表示来单提醒 2表示客户催单  
map.put("orderId", ordersDB.getId());  
map.put("content", "订单号: " + outTradeNo);

```
**流程：** 配置好 `WebSocket` 配置类和 `webSocketServer` 工具类 -> 封装成前端需要的 `JSON` 数据 -> 调用 `webSocketServer` 工具类方法。

### 设置名字：

**方法：** `void setName(String name)`  或 `Thread(Runnable target, Stirng name)` 【可传入线程的任务对象】 或 `Thread(Stirng name)`
① 如果没有给线程设置名字，则线程会有默认名字： `Thread-X` （X序号，从0开始，每创建一个线程对象就会依次递增）；

② 如果要给线程设置名字，可以用set方法进行设置，也可以在构造方法中设置。

### 设置优先级：

**方法：** `setPriority(int newPriority)`
① 在Java中采用的是抢占式调度线程的执行【随机执行线程】；

② 线程的优先级越大抢占到CPU的概率越高，线程优先级区间：1~10（默认为5）；

③ 优先级不是绝对的，仅仅只是提升了优先概率。

### 守护线程【备胎线程】：

**方法：** `final void setDaemon(boolean on)`
![img](/Draven_Note_Images/Java/23.多线程/23.%20多线程_守护线程：_08.png)
**注意：** 为true的时候就是守护线程。说白了就是即使备胎线程还没执行完，但是非备胎线程已经执行完了，那么备胎线程就会陆续结束。
### 出让/礼让线程：

**方法：** `public static void yield()`
``` java
@Override  
public void run() {  
    for (int i = 0; i < 100; i++) {  
        System.out.println("MyThread is running: " + getName() + "-" + i);  
        // 表示出让当前CPU的执行权, 尽可能的让线程执行均匀一点  
        Thread.yield();  
    }  
}
```

### 插入/插队线程：

**方法：** `public static void join()`
``` java
static void main() throws InterruptedException {  
        MyThread t = new MyThread("土豆");  
        t.start();  
  
        // 表示把t这个线程，插入到当前线程之前。  
        // t: 土豆  
        // 当前线程: main线程  
        t.join();  
  
        // 执行在main线程当中  
        for (int i = 0; i < 100; i++) {  
            System.out.println("main线程: " + i);  
        }  
    }
```
**作用：** 只有该线程执行完了main线程才会开始执行，可避免被main线程抢占CPU。

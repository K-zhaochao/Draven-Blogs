#### `IP` 在Java中的应用：

`InetAddress` 类：
``` java
static void main() throws UnknownHostException {  
    InetAddress ip = InetAddress.getByName("192.168.56.1");  
  
    System.out.println(ip);  
    System.out.println(ip.getHostAddress());  
    System.out.println(ip.getHostName());  
}
```

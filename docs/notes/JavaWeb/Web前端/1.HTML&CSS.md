### CSS选择器：
<img src="/Draven_Note_Images/JavaWeb/Web前端/CSS选择器：_07.png" alt="img" />

+ **id 选择器**：是元素的「唯一标识」（类似身份证号），全局范围内只能对应一个元素，用于精准定位单个元素；
+ **类选择器**：是元素的「分类标签」（类似职业标签），可给多个元素贴同一标签，用于批量复用样式 / 行为。

**优先级：**id选择器 > 类选择器 > 元素选择器

# `slot-scope="scope"` （Element UI 表格场景）

***

## 1. 核心本质

这是 **Vue2 作用域插槽** 在 **Element UI **`el-table`** 表格列自定义渲染** 中的典型用法：

* 子组件：`el-table-column`（Element 封装的表格列组件）
* 父组件：业务代码（写 `<template>` 的位置）
* 作用：**子组件（**`el-table-column`**）将当前行数据、列信息等传递给父组件插槽**，父组件通过 `slot-scope="scope"` 接收所有传递的数据。

***

## 2. 代码逐行拆解

```vue
<template slot-scope="scope">
  <!-- Element UI 时间图标 -->
  <i class="el-icon-time"></i>
  <!-- 用 scope.row.date 取当前行的 date 字段，加左边距 -->
  <span style="margin-left: 10px">{{ scope.row.date }}</span>
</template>

```

### 关键说明

| 代码 | 含义 |
| --- | --- |
| `slot-scope="scope"` | 定义**自定义变量**`scope`，接收子组件（`el-table-column`）传递的所有数据（可自定义命名，如`slot-scope="rowData"`） |
| `scope.row` | `el-table-column` 默认传递的**当前行完整数据对象**（包含接口返回的该行所有字段，如`date`/`name`/`id`） |
| `scope.row.date` | 从当前行数据中取出`date`字段，用于渲染 |
| `scope.column`/`scope.$index` | 额外传递的属性：当前列配置、当前行索引（从0开始） |

***

## 3. 子组件（el-table-column）内部逻辑（简化版）

Element 已经帮我们封装好了插槽数据绑定，内部逻辑类似：

```vue
<!-- el-table-column 子组件内部 -->
<template>
  <div class="el-table__cell">
    <!-- 子组件主动给插槽绑定 row、column、$index 三个属性 -->
    <slot :row="currentRow" :column="columnConfig" :$index="rowIndex"></slot>
  </div>
</template>

```

因此父组件的 `scope` 本质是 `{ row: 行数据, column: 列配置, $index: 索引 }` 这个对象。

***

## 4. 完整上下文示例（Vue2 + Element UI）

```vue
<template>
  <el-table :data="tableData" border>
    <el-table-column prop="name" label="姓名" width="120"></el-table-column>
    <!-- 自定义日期列 -->
    <el-table-column prop="date" label="日期" width="180">
      <!-- 图中代码的完整位置 -->
      <template slot-scope="scope">
        <i class="el-icon-time"></i>
        <span style="margin-left: 10px">{{ scope.row.date }}</span>
      </template>
    </el-table-column>
  </el-table>
</template>
<script>
export default {
  data() {
    return {
      // 表格数据源，每一行的 date 会通过插槽传递给 scope.row
      tableData: [
        { date: "2026-04-02", name: "张三" },
        { date: "2026-04-01", name: "李四" }
      ]
    };
  }
};
</script>

```

***

## 5. 关键补充

1. **命名自由**：`scope` 不是固定关键字，可自定义为 `slot-scope="data"`，后续用 `data.row.date` 即可。
2. **Vue3 迁移**：Vue3 废弃 `slot-scope`，替换为 `v-slot` 语法，Element Plus 中写法为：

```vue
<template #default="scope">
  <i class="el-icon-time"></i>
  <span>{{ scope.row.date }}</span>
</template>

```

1. **适用场景**：不仅限于表格，所有需要「子组件给父组件插槽传数据」的 Vue2 场景都可使用，表格是最常见的业务场景。

***

## 核心总结

* 这个绑定的本质：**UI组件（el-table-column）主动给插槽传当前行数据，父组件用slot-scope接收并自定义渲染**
* 核心价值：在不修改子组件（UI组件）源码的前提下，灵活自定义列的样式/结构，同时拿到行数据。

***

# slot-scope="scope" （Vue2 作用域插槽）

## 1. 核心定义

Vue2 专属**作用域插槽语法**，实现**子组件向父组件插槽传递数据**，父组件通过 `scope` 接收子组件绑定的所有数据。

## 2. 核心规则

1. **子组件**：在 `<slot>` 标签上**绑定数据**（`:自定义属性="数据"`）
2. **父组件**：在插槽元素上写 `slot-scope="scope"`
3. `scope`：**自定义对象**，承载子组件插槽绑定的所有属性

## 3. 极简示例

### 子组件（ItemBox.vue）

```vue
<template>
  <div>
    <!-- 子组件绑定数据：id、name -->
    <slot :id="1" :name="测试数据"></slot>
  </div>
</template>

```

### 父组件使用

```vue
<template>
  &lt;ItemBox&gt;
    <!-- 接收子组件数据，渲染使用 -->
    <div slot-scope="scope">
      ID：{{ scope.id }}，名称：{{ scope.name }}
    </div>
  </ItemBox>
</template>

```

## 4. 关键补充

* `scope` 可自定义命名（如 `slot-scope="data"`）
* Vue3 已废弃，替换为 `v-slot="scope"`（简写 `#default="scope"`）

***

### 总结

1. 作用：子组件给父组件插槽传数据
2. 用法：子绑数据、父用 `slot-scope` 接收
3. 本质：`scope` 是子组件插槽数据的集合对象

# 附录2：关键字

**关键字 (Keywords)** 是 Python 语言的“基石”。它们是 Python 内部保留的单词，具有特殊的语法含义，你**不能**用它们来命名变量、函数或类。Python 3.x 大约有 35 个关键字。为了让你记忆更深刻，我将它们分为 **6 大核心类别** 来进行系统讲解。

***

### 第一类：布尔逻辑与真值 (Boolean Logic & Values)

这部分是编程逻辑的基础，就像数学中的逻辑代数。

**关键字列表：**\
`True`, `False`, `None`, `and`, `or`, `not`, `is`

#### 1. 核心讲解

* `True**` / `**False`: 布尔类型的两个值，真与假。注意首字母大写。
* `None`: 代表“空”或“无”。它不是 0，也不是空字符串，就是一个特殊的空对象。
* `and**` / `**or**` / `**not`: 逻辑运算符。
  * $ A \land B $ (`A and B`)：两者都为真才为真。
  * $ A \lor B $ (`A or B`)：只要有一个为真即为真。
  * $ \neg A $ (`not A`)：取反。
* `is`: 判断两个对象是否是**同一个**（内存地址是否相同），而不仅仅是值相等。

#### 2. 代码演示

```python
# 伪代码/示例
x = True
y = False
z = None

# 逻辑运算
if x and (not y):
    print("X是真，且Y不是真")  # 会执行

# `is` vs `==` 的区别
a = [1, 2, 3]
b = [1, 2, 3]
print(a == b)  # True (值相等)
print(a is b)  # False (它们是两个不同的列表对象，内存地址不同)
```

### 第二类：流程控制 (Control Flow)

这部分决定了代码执行的顺序和方向。

**关键字列表：**\
`if`, `elif`, `else`, `for`, `while`, `break`, `continue`, `in`

#### 1. 核心讲解

* `if**` / `**elif**` / `**else`: 条件判断。如果...否则如果...否则...
* `for`: 遍历循环（通常用于列表、字符串等）。
* `while`: 条件循环（只要条件满足就一直跑）。
* `in`: 成员运算符。检查一个元素是否存在于列表中。
* `break`: **彻底终止**当前循环。
* `continue`: **跳过本次**循环的剩余部分，直接进入下一次循环。

#### 2. 代码演示

```python
# 场景：老师点名，遇到调皮的学生跳过，遇到下课铃声停止

students = ["小明", "调皮鬼", "小红", "铃声", "小刚"]

for s in students:  # s 依次代表列表中的每个元素
    if s == "铃声":
        print("下课了，不点了！")
        break  # 终止整个循环，小刚不会被点到
  
    if s == "调皮鬼":
        print("跳过调皮鬼...")
        continue  # 跳过本次print，直接进入下一个人
      
    print(f"点名：{s}")

# 输出：
# 点名：小明
# 跳过调皮鬼...
# 点名：小红
# 下课了，不点了！
```

### 第三类：函数与数据定义 (Functions & Data Definitions)

这部分用于构建程序的功能模块。

**关键字列表：**\
`def`, `return`, `lambda`, `pass`, `yield`, `class`

#### 1. 核心讲解

* `def`: 定义一个函数 (Define)。
* `return`: 从函数中返回结果，并结束函数。
* `class`: 定义一个类 (面向对象编程的核心)。
* `pass`: 占位符。什么都不做，用于保持语法结构完整（比如先写个空函数占坑）。
* `lambda`: 定义匿名函数（一种简写的单行函数）。数学上表示为 $ \lambda x. f(x) $。
* `yield`: 生成器专用。类似于 `return`，但它会“暂停”函数并保存状态，下次调用时从暂停处继续执行。

#### 2. 代码演示

```python
# 1. 普通函数
def add(a, b):
    return a + b

# 2. Lambda 函数 (数学公式 f(x) = x^2)
square = lambda x: x * x
print(square(5))  # 输出 25

# 3. 占位符
class FutureFeature:
    pass  # 暂时还没想好写什么，先放个pass防止报错

# 4. yield (生成器) - 就像发牌员，发一张牌停一下，而不是一次把整副牌扔给你
def card_dealer():
    yield "红桃A"
    yield "黑桃K"

dealer = card_dealer()
print(next(dealer)) # 输出：红桃A
# 此时函数暂停在第一行 yield 之后
```

### 第四类：变量作用域与模块 (Scope & Modules)

这部分管理变量在哪里可见，以及如何引用外部代码。

**关键字列表：**\
`import`, `from`, `as`, `global`, `nonlocal`

#### 1. 核心讲解

* `import**` / `**from`: 导入外部库或模块。
* `as`: 起别名（Alias）。比如库名太长，起个短名字。
* `global`: 在函数内部声明变量为**全局变量**（即修改函数外面的那个变量）。
* `nonlocal`: 在嵌套函数中，修改外层（非全局）函数的变量。

#### 2. 代码演示

```python
import math as m  # 导入math库并起别名m

count = 0

def change_global():
    global count  # 告诉Python我要改的是外面那个count
    count = 10

change_global()
print(count)  # 输出 10 (如果没有 global 关键字，这里还是 0)
```

### 第五类：异常处理与资源管理 (Exception & Context)

这部分让程序更健壮，不容易崩盘。

**关键字列表：**\
`try`, `except`, `finally`, `raise`, `assert`, `with`

#### 1. 核心讲解

* `try**` / `**except**` / `**finally`: 尝试运行代码 / 如果出错怎么办 / 无论是否出错最后都要做的事。
* `raise`: 主动抛出一个错误（比如用户输入不合法，你主动报错）。
* `assert`: 断言。用于调试，判断某个条件是否为真，如果为假则报错。
* `with`: 上下文管理器。通常用于文件操作，它会自动帮你关闭文件，非常安全。

#### 2. 代码演示

```python
# 资源管理 (with)
# 也就是：打开文件 -> 操作 -> 自动关闭(哪怕中间出错了)
with open("test.txt", "w") as f:
    f.write("Hello")

# 异常处理
def divide(a, b):
    try:
        result = a / b
    except ZeroDivisionError:
        print("错误：除数不能为0")
    else: # 只有 try 块内的代码完全执行成功（没有抛出任何异常），才会执行 else 块的代码。
        print(f"结果是：{result}")
    finally: 
        print("计算结束")  # 无论是否出错都会打印（哪怕 try 块里有 return 语句，也会先执行 finally 再返回）

divide(10, 0)
```

### 第六类：其他与高级特性 (Others)

一些特定的、较少见或高级的用法。

**关键字列表：**\
`del`, `async`, `await`

#### 1. 核心讲解

* `del`: 删除对象（变量、列表中的元素等）。
* `async**` / `**await`: 异步编程专用。
  * `async def`: 定义一个异步函数（协程）。
  * `await`: 等待一个异步操作完成（非阻塞）。这在爬虫或高并发服务器中很常见。

#### 2. 代码演示

```python
# del 演示
my_list = [10, 20, 30]
del my_list[1]  # 删除索引为1的元素
print(my_list)  # 输出 [10, 30]
```

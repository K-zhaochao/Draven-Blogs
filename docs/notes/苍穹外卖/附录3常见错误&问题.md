# 附录3 常见错误&问题

### 主键返回

在**`Service`**层的操作时，**表2**的操作需要用到**表1**生成的主键时，需要等**表1**生成主键之后再获取。主键返回无需在**`Service`**层新创建对象来接受，该返回的主键会自动填充到传入**`Mapper`**层的对象中，直接获取即可。

**举例：**

```java
@Transactional
public void saveWithDish(SetmealDTO setmealDTO) {
Setmeal setmeal = new Setmeal();
    BeanUtils.copyProperties(setmealDTO, setmeal);

    //向套餐表插入数据
    setmealMapper.insert(setmeal);

    //获取生成的套餐id
    Long setmealId = setmeal.getId();

    List&lt;SetmealDish&gt; setmealDishes = setmealDTO.getSetmealDishes();
    setmealDishes.forEach(setmealDish -> {
        setmealDish.setSetmealId(setmealId);
    });

    //保存套餐和菜品的关联关系
    setmealDishMapper.insertBatch(setmealDishes);
}
```

### 分页查询

在一些界面的分页查询中，既需要用到该界面所需的**表1**，又因为该**表1**中只有另一个**表2**属性的id，没有**表2**的名字或者其他属性，在查询的时候就需要用到左外连接的方式来获取，在返回的时候，要记得给所需要的**表2**属性进行重命名，不然无法映射。

**举例：**

```xml
<select id="pageQuery" resultType="com.sky.vo.SetmealVO">
  select s.*,c.name categoryName
  from sky_take_out.setmeal s left join sky_take_out.category c on c.id = s.category_id
  <where>
    <if test="name!=null and name!=''">s.name = #{name}</if>
    <if test="categoryId!=null">s.category_id = #{categoryId}</if>
    <if test="status!=null">s.status = #{status}</if>
  </where>
</select>
```

### 前端请求数据

阅读接口文档的时候，要注意观看前端是否在请求中给后端需要的东西，如果没有给且在数据表中需要该数据都话，后端需要自行补全。

**举例：前端给出的数据中并没有套餐的ID，因此后端需要自行补全套餐菜品关系表中的套餐ID**

![img](/Draven_Note_Images/苍穹外卖/附录3常见错误&问题/附录3%20常见错误&问题_前端请求数据_01.png)

```java
/**
 * 修改套餐
 * @param setmealDTO
 */
@Transactional
public void update(SetmealDTO setmealDTO) {
    // 数据准备
    Setmeal setmeal = new Setmeal();
    BeanUtils.copyProperties(setmealDTO, setmeal);
    List&lt;SetmealDish&gt; setmealDishes = setmealDTO.getSetmealDishes();
    Long setmealId = setmealDTO.getId();
    for (SetmealDish setmealDish : setmealDishes) {
        setmealDish.setSetmealId(setmealId);
    }

    // 更新套餐数据
    setmealMapper.update(setmeal);

    // 更新套餐与菜品的关系数据
    setmealDishMapper.deleteBySetmealId(setmealId);
    setmealDishMapper.insertBatch(setmealDishes);
}
```

### 请求问题

在用**`HttpClient`**进行请求的时候，要注意其返回的值是一个JSON格式，需要通过`alibaba.fastjson`里面的**`JSONObject`**工具类进行类的转换，再通过**`JSONObject`**的`getString(属性名)`来获取要查询的数据。

**举例：**

```java
// 获取请求之后返回的JSON数据
String json = HttpClientUtil.doGet(WX_LOGIN, paramMap);

// 从该JSON数据中找到对应的属性
JSONObject jsonObject = JSONObject.parseObject(json);
String openid = jsonObject.getString("openid");
```

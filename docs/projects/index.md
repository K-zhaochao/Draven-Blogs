# 项目实战

> 纸上得来终觉浅，绝知此事要躬行。

学习的最终目的是解决问题。这个板块记录了我参与或主导的项目，从需求分析到技术选型，从编码实现到踩坑修复，尽量完整地还原整个过程。

---

::: info 说明
每个项目都会包含：背景介绍、技术栈、核心设计思路、以及实际开发中遇到的问题和解决方案。

不是每一行代码都值得写，但每一个坑都值得记住。
:::
